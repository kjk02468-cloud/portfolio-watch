1. 투자 목적 : 
가. 26년 6월부터 원금 3,000만원과 매월 100만원+@(보너스, 성과금 등)을 투자해 29년 6월 5억+을 만드려고 함

2. 투자 방식 : 현재 회사가 속해있는 테마(단일 혹은 다중) 중 어느 단계에 와있는지에 따라 5단계로 구분하여 비중을 달리함.
가. 투자 5단계
  1) 단계 별 구분
  - 0단계(0%) : 포트폴리오 편입 준비 단계 / 단순 상상의 영역(1단계 신호 어떤 것도 X)
  - 1단계(20%) : 수주 신호 등장 / 가격 바닥권 / 실적 아직 
  - 2단계(40%) : 고객, 출하 확인 / 급등 시작
  - 3단계(25%) : 가이던스 상향 / 런업 / 촉매 임박 
  - 4단계(15%) : 실적 폭발 / 급등권 
    * 4-A단계(+@%, 세속적 대장) : 세속적 대장(매분기 새 촉매·추정치 계속 상향): 4단계여도 일부는 스케줄 무시하고 ride. 추정치 상향 멈출 때 컷.
    * 4-B단계(15%, 단발 종목) : 단발 종목(촉매 1개): 스케줄대로 트림 (15%까지 축소)
  2) 단계 내 테마 및 종목 구분 주기 및 방법 : (CLAUDE 보완 필요)
  - 탐색 주기 :
    * 상시(이벤트 발생 시): 실적·수주·계약·가이던스 뉴스 뜨면 즉시 → 해당 종목 단계 재판정
    * 주 1회(주말): 보유 5종목 킬신호 스캔 + 추정치 방향 점검
    * 월 1회(첫 거래일): 적립금 투입 + 0단계 후보 발굴
    * 분기 1회(실적시즌): 테마 전체 재점검, 밸류체인 새 병목 탐색 
  - 방법 : 
     * 수주/백로그 (1→2) : 실적발표 book-to-bill, 8-K 계약공시
     * 고객·출하 확정 (2→3) : 어닝콜 코멘트, 신규고객 PR
     * 가이던스 상향 (3→4)실적 가이던스, 추정치 상향추정치 상향 멈춤 (4→매도)컨센 EPS 리비전 추세

나. 월별 or 분기별 단계 리레이팅 및 회전형 리밸런싱 실시
   1) 1단계 종목이 2단계로 가면 : 종목 추가 OR 비중 추가
   2) 2단계 종목이 3단계로 가면 : 종목 제외 X / 비중 감소
   3) 3단계 종목이 실적 발표 후 4단계로 갈지 결정해야 할 때 아래 기준 활용
     - 단발 종목(촉매 1개): 스케줄대로 트림 (15%까지 축소)
     - 세속적 대장(매분기 새 촉매·추정치 계속 상향): 4단계여도 일부는 스케줄 무시하고 ride. 추정치 멈출 때 컷.

   4) 포트폴리오 제외 : 1~3단계 판단 MISS 확인 시 OR 시나리오 파괴할만한 SELL 뉴스 터질 때
   5) 포트폴리오 편입 : 1~3단계 새로운 뉴스 터졌을 때 OR 0단계의 가파른 성장 시  
   6) 현금 발생 시 : 1~2단계 종목만 추가 매수, 3단계 종목은 어차피 올라있거나 오를 예정
